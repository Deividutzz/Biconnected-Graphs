# Biconnected Graphs



This c++ source solves the problem of graph biconnectivity,

such as determining the components of the graph that are biconnected and more.



##### **How it works:**



There is an in.txt file that you have to open and insert the input there.

Then double-click the exe file to run the program.

If the input is entered correctly, then in the out.txt file it will appear the results.



##### **Here's the structure of the input:**



2 natural numbers: (n,m); One representing the number of verticies and the other the number of edges.

Then m ( == number of edges inserted before) pairs of (x,y) where x has edge with y.
